# ⌚ Watches-Earpods-Store-PK

An online storefront for **watches, earbuds, and smartphones** built using **HTML, CSS, and JavaScript**. The website features a modern responsive design, animated hero section, interactive product categories, product details page, and cash-on-delivery ordering for customers in Pakistan.

---

## 📌 Features

- 🛍️ Modern online storefront
- ⌚ Smartwatch collection
- 🎧 Earbuds collection
- 📱 Mobile phone collection
- 🎉 12.12 Sale section
- ⭐ Top Products section
- 🌌 Animated parallax hero section
- 🎨 Interactive flowing category menu
- 🖼️ Product image gallery
- 💰 Product pricing with discounts
- 🛒 Buy Now functionality
- ➕ Add to Cart button (UI)
- 📦 Order Tracking section
- ℹ️ About Us section
- 📱 Responsive design
- 💵 Cash on Delivery across Pakistan

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Google Fonts (Poppins)

## 🚀 How to Run

1. Clone the repository

```bash
git clone https://github.com/1256coder/Watches-Earpods-Store-PK.git
```

2. Open the project folder.

3. Double-click **index.html** or open it in your preferred web browser.

No additional software or installation is required.
